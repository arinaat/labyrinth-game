Лабиринт сокровищ

Консольная текстовая игра на Python.
Игрок исследует лабиринт, собирает предметы, решает загадки и пытается открыть сундук с сокровищами.

Требования:
- Python 3.11+
- Poetry

Установка:
make install

Запуск игры:
make project

Проверка кода:
make lint

Команды в игре:
look        — осмотреть комнату
inventory  — инвентарь
go <dir>   — перейти (north, south, east, west)
take <item>— взять предмет
use <item> — использовать предмет
solve      — решить загадку
open       — открыть сундук
quit       — выйти из игры

Победа:
Открытие сундука с сокровищем.

Поражение:
Здоровье игрока опускается до нуля.
